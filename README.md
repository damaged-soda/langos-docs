# langos-docs

langos 项目的高层文档库，包含愿景、治理、自举说明等背景资料；代码仓仍在 `../langos/`。

## 阅读顺序
1) `repos/INDEX.md` 了解项目列表（当前只有 langos）。  
2) `repos/langos/overview.md` 获取项目概览与目录地图。  
3) 视需要阅读：愿景 `vision.md`、meta 介绍、治理自举说明、specs 目录提示等。  
4) 执行/编码时，请回到代码仓按 `runtime/startup.md`、`guidelines.md`、`runtime.md`、`protocols/index.yaml` 进行。

## 目录骨架（可按需扩展）
- `repos/langos/`：langos 项目的背景文档。  
- `specs/`、`projects/`、`adr/`、`domain/`、`archive/`：预留目录，按需补充。  

> 提醒：AI 运行时的“必读最小集”在代码仓；本仓用于理解设计思路与演进记录。
