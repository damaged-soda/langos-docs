# specs 目录说明

存放跨仓或通用的需求规格文档；单个仓库专属的规格可放在 `specs/<repo>/` 子目录（例如 `specs/langos/` 存放内核相关规格）。

推荐命名：`YYYYMM-short-name-spec.md`，必要时按主题或仓库分子目录。

当前可参考的规格：
- `specs/langos/langos-runtime-docs-decoupling-spec.md`
- `specs/langos/202512-langos-docs-classification-spec.md`

新增规格时请简述背景/目标、范围/非范围、输入输出、确认点与验收标准，遵守 `runtime/conventions.md`。***
